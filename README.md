# PlurkOnWatch

To view plurk on Apple Watch.

## Device that you need
- iPhone
- Apple Watch

## How to use
1. Install APP on your iPhone & Apple Watch
2. Login from iPhone & keep APP on Apple Watch running
3. It should work now.

## Implemented

- [x] 河道 (public, only mine)
- [x] Response count
- [x] See response
- [x] My profile 
- [ ] Response post
